        \section{\textsc{Research Backgrounds}}
        My research interests are computer architecture, parallel I/O and
        compiler techniques. I did some investigation on the hardware data
        prefetching algorithms for the last level cache, for which we wrote a
        cache simulator based on \textit{pintool}. I also took part in a
        event-driven simulator project at Intel Labs China. Now I focus myself
        on the I/O storage options in cloud platforms such as EC2. Our
        priliminary work shows its potential but how to automating the process
        is still to be examined. I believe the compiler can be employed for
        the future work, so I start to do some research in the program slicing
        too.
